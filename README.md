# EduCode Editor

Простой редактор кода на Python со встроенным терминалом и подсветкой синтаксиса.

## Возможности

- Подсветка синтаксиса Python (ключевые слова, строки, комментарии, числа)
- Встроенный терминал для запуска команд
- Запуск Python-скриптов прямо из редактора
- Открытие, создание и сохранение файлов

## Установка и запуск

```bash
git clone https://github.com/MrZero-c/EduCodeEditor.git
cd EduCodeEditor
python main.py
